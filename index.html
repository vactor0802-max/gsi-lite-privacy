Privacy Policy

Effective Date: April 1, 2026

Summary: GSI Lite does not collect, transmit, or store any personal data on external servers. All data stays on your device.

1. Information We Collect

GSI Lite does not collect any personal information.

  • No account registration or login required
  • No name, email, phone number, or address collected
  • No analytics or tracking services used
  • No advertising SDKs or ad networks integrated

2. Data Stored on Your Device

The following data is stored locally on your device only using AsyncStorage:

  • Recipe history: Previously analyzed recipes (ingredients and amounts)
  • My Seasoning data: Custom seasoning profiles you create
  • Language preference: Korean or English setting
  • Onboarding status: Whether you have completed the intro screen

This data is never transmitted to any server. Uninstalling the app permanently deletes all stored data.

3. Camera and Photo Library

GSI Lite requests camera/photo permissions only for OCR recipe scanning.

  • OCR processing is performed entirely on-device using ML Kit (offline)
  • No images are uploaded, transmitted, or stored externally
  • Images are processed in memory and discarded immediately after text extraction

4. Third-Party Services

GSI Lite uses the following third-party components, none of which collect user data:

  • Expo / React Native: App framework (no telemetry in production builds)
  • rn-mlkit-ocr: On-device text recognition (no network calls)
  • Expo Updates (OTA): Checks for app updates only; no user data transmitted

5. Children's Privacy

GSI Lite does not knowingly collect information from children under 13. The app contains no age-restricted content.

6. Data Security

Since all data remains on your device and no network transmission occurs, the risk of data breach is minimal. Data is protected by your device's own security measures.

7. Changes to This Policy

We may update this Privacy Policy. Changes will be reflected on this page with an updated effective date.

8. Contact

If you have questions about this Privacy Policy:

  • Email: gsilite.app@gmail.com

개인정보처리방침

시행일: 2026년 4월 1일

요약: GSI Lite는 개인정보를 수집, 전송, 외부 저장하지 않습니다. 모든 데이터는 사용자 기기에만 저장됩니다.

1. 수집하는 정보

GSI Lite는 어떠한 개인정보도 수집하지 않습니다.

  • 회원가입이나 로그인 불필요
  • 이름, 이메일, 전화번호, 주소 등 수집하지 않음
  • 분석 도구나 추적 서비스 미사용
  • 광고 SDK나 광고 네트워크 미탑재

2. 기기에 저장되는 데이터

다음 데이터는 AsyncStorage를 통해 사용자 기기에만 저장됩니다:

  • 레시피 기록: 이전에 분석한 레시피 (재료 및 양)
  • 내 조미료 데이터: 사용자가 등록한 조미료 프로필
  • 언어 설정: 한국어 또는 영어
  • 온보딩 상태: 인트로 화면 완료 여부

이 데이터는 어떤 서버로도 전송되지 않습니다. 앱 삭제 시 모든 데이터가 영구 삭제됩니다.

3. 카메라 및 사진 접근

GSI Lite는 OCR 레시피 스캔 용도로만 카메라/사진 권한을 요청합니다.

  • OCR 처리는 ML Kit을 사용하여 기기 내에서만 수행 (오프라인)
  • 이미지는 외부로 업로드, 전송, 저장되지 않음
  • 텍스트 추출 후 이미지는 즉시 메모리에서 삭제

4. 제3자 서비스

GSI Lite가 사용하는 제3자 구성요소는 사용자 데이터를 수집하지 않습니다:

  • Expo / React Native: 앱 프레임워크 (프로덕션 빌드에서 텔레메트리 없음)
  • rn-mlkit-ocr: 기기 내 텍스트 인식 (네트워크 호출 없음)
  • Expo Updates (OTA): 앱 업데이트 확인만 수행; 사용자 데이터 미전송

5. 아동 개인정보

GSI Lite는 13세 미만 아동의 정보를 고의로 수집하지 않습니다. 앱에 연령 제한 콘텐츠가 없습니다.

6. 데이터 보안

모든 데이터가 기기에만 저장되고 네트워크 전송이 없으므로, 데이터 유출 위험이 최소화됩니다. 데이터는 기기 자체 보안 기능으로 보호됩니다.

7. 방침 변경

본 개인정보처리방침은 변경될 수 있으며, 변경 시 이 페이지에 시행일과 함께 반영됩니다.
